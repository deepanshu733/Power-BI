This Power BI practice dashboard analyzes sales performance across different regions, product categories, and unit prices. The report provides interactive visualizations to identify sales trends and compare regional performance. Key metrics such as total sales, category-wise distribution, and pricing insights have been explored using various charts and filters. This project helped strengthen my understanding of data visualization and dashboard design in Power BI. It also improved my skills in creating interactive reports and deriving business insights from data. The dashboard demonstrates how data can be transformed into meaningful and actionable information.
